Sobre a Linguagem HTML

!DOCTYPE HTML

O Doctype funciona como documento para criar estruturas de um site e define a versão do HTML.

html

O HTML é uma Linguagem de Marcação de Textos que define a estrutura do documento.

head

Não é exibida no navegador, mas pode conter metadados, link para vincular como um Arquivo CSS e título como o nome do site.

body

O Body é um corpo do HTML que pode ser exibido nos navegadores e pode ser visivel para o usuário.
